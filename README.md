# Fetch-api
**Interactive Fetch User Data**
A simple, interactive webpage that fetches user data from the JSONPlaceholder API using JavaScript's Fetch API. Users can view, edit, add, and delete data locally (changes are not saved to the API).


**Features**
Fetches and displays user data (name, email, address) from JSONPlaceholder.

*Local editing: Click to edit text inline (persists in browser via localStorage).

*Add new users locally with a form.

*Delete users with a button.

*Reload button to refetch original data.

*Error handling with retry for network issues.

*Responsive design with a gradient background.


**Technologies Used**
HTML
CSS
JavaScript (Fetch API, localStorage)


**How to Run**
Clone or download this repository. 

Open index.html in a web browser (e.g., Chrome).

The page will automatically fetch and display data.
